Proof Of Concept exploring different cryptocurrency conversion methods; Shapeshift, Changelly & Evercoin.

In production, the Changelly widget will be used as it removes liability to MyBit since Changelly are directly handling funds not MyBit making API calls on behalf of the user.

go root directory
1) npm init
2) npm install
3) node index.js
